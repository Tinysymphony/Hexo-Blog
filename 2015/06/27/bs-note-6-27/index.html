nodejs 压缩文件

可以使用操作系统的命令，为防止操作系统依赖，使用npm的包。

选择了几个包最后用的是archive

git地址:https://github.com/archiverjs/node-archiver

npm install archive即可（比较慢的话使用淘宝的npm镜像）

使用流程是：

var fs = require(‘fs’);
var archiver = require(‘archiver);
var archive = archiver.create(‘zip’, {}); //确定格式
var output = fs.createWriteStream(zipPath); //创建zip，这个zipPath其实是路径+文件名
archive.pipe(output); //管道
archive.bulk([
{src: [‘xxx’]},
{expand: true, cwd: ‘xxx’, src:[‘xxx’], dest: ‘xx’},
xxx
]);
archive.finalize();

路径可以使用相对或绝对路径，src: [‘pp/**’]表示压缩当前目录下pp文件夹内所有内容。

cwd: ‘xxx’表示使用路径xxx，src [‘xxx’]表示使用之前那个路径下面的xxx路径，dest: ‘xx’ 表示在压缩包的路径头。

目标是让用户能下载网页结果，以zip包的形式。

前端js不能下载文件，所以使用添加透明表单发送post请求接收数据，后端组装压缩包传送回来。

后端写在/ExMint/routes/editor.js中，路由为/editor/download
