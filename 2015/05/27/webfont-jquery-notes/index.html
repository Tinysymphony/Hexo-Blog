WebFont & jquery notes

Fronter

在看一些前端代码时发现有个很奇怪的用法：

.CloseButton:before {
content: “\e005”;
}

虽然知道:before和:after相当于添加子节点（伪元素），但这个content和\xxx之类的东西就不知道哪儿冒出来的。

后来找了找唯一可能有关联的是头上的：

@font-face {
    font-family: ‘icomoon’;
    src:url(‘./fonts/icomoon.eot’);
    src:url(‘./fonts/icomoon.eot?#iefix’) format(‘embedded-opentype’),
        url(‘./fonts/icomoon.woff’) format(‘woff’),
        url(‘./fonts/icomoon.ttf’) format(‘truetype’),
        url(‘./fonts/icomoon.svg#icomoon’) format(‘svg’);
    font-weight: normal;
    font-style: normal;
}

看了下这里所引入的字体并不是正常的东西，里面的“字母”是一套logo，然后查了下这类东西叫webfont.
将矢量图标做成“字体”，有更好的兼容性（上面使用了不同格式增强对浏览器的支持）。不需要css切图获取icon，矢量也可以随意调整大小。

引入这个网页图标字体后，可以用字体的方式调用

.icon{
        font-family: ‘MyFont’; / 使用方法如同字体 /
        speak:none; / 使屏幕阅读器不读这个字符 /
        font-size:14px;
        font-variant:normal;
        font-weight:normal;
        text-transform: none;
        …
}

/ 如果在头部加icon /
.icon_open:before{
content:”\e001”; / 要用到的icon编号 /
…
}

 

 

初学jquery，因为想做B/S作业的网站。
目前也只会用几个简单的操作，连原生js都没好好学真是太急躁。

记一下好用的几个东西。

 

1.控制滚轮移动到特定元素

$(“#your_container”).scrollTop()可以直接设置滚轮的位置

$(“#your_target”).offset().top能获取到目标到当前点的相对位置

所以只要设置scrollTop属性的数值为：

target.offset().top - container.offset().top + container.scrollTop()

当然，.offset()前面是选择器获取的元素

 

2.js辅助自适应

css自适应一般靠百分比显示，auto!important 或者 min-xxx这样的形式控制

有时候处理一些坑可以用$(window).resize(function(){ … })这样的形式处理

 

3.setTimeout玄学

目前处于想到什么写什么，都去尝试的状态。
有次想实现两个div的自动对齐（外面嵌个div都设置100%啥的应该也能解决问题）

但是想让jquery干这个事，ready和resize里都加了操作，但初次加载时就是不对齐。
不太清楚ready的机制，ready里面的那行对齐代码貌似没有执行，其他有效的代码基本都是在一些事件里面的，莫非是这个原因？

然后试着用setTimeout，在ready后0s/10s执行这段代码，发现初次加载就对了，不过感觉挺蠢的Orz…
