Raspberry Pi ゲット！

  • rpi
Embedded System

寝室三人抽签决定三块板子的归属..说好的ps3对战呢

您的板子——树莓派——已经上线..

8G SD 卡上已经装了系统，插入就好
通过串口连接PC..白绿黑三线分别对I4,I5,GND..推荐使用红线连接Power口供电，充电口配合三棒子的充电器经常接触不良。

因为PC是Ubuntu，所以安装了kermit用于接受串口输入

sudo apt-get install kermit

在不接电的情况下可以在/dev/目录下ls tty*以确定设备的名称，一般是ttyUSB0 或者 ttyUSB1
而后编辑主目录下的配置文件，修改要对接的设备名..

vim ~/.kermrc

这里只修改一行:set line /dev/ttyUSB0
之后在连上红线后，在pc terminal中输入：
sudo kermit （不造为什么一定要root权限求解释）
然后按下c即可接收来自板子的启动信息，强行刷屏

用hdmi连接寝室显示器也可以看到boot信息，默认不是启动图形界面，登陆后用命令：

sudo raspi-config

即可修改…（板子完全启动后，再连接显示器视频信号可能无法输出Orz）

推荐用PC调试板子，使用rasbian的图形界面需要很大耐心.

系统和debian类似，命令基本通用，可以用lscpu/df/free/dd/uname/ifconfig/lsblk等命令查看一些配置、
比如这样…

联网以及php/mysql/apache2

基本命令可得，这里不罗嗦。

遇到最大的问题是..mysql无法从官方源获取，一直404，代理抢救无效。

尝试载源文件编译安装….按一些博客的说法需要使用cmake..（但在cmake安装时也会报错啊摔！）

所以，改源吧少年。

sudo vim /etc/apt/source.list

号注释掉官源，改为国内镜像站点，比如我浙，某科大，某华……

以下均可，sudo apt-get update后无压力。

 

怎么联网？

拉网线连寝室路由DHCP一发…（l2tp路由大法好）

 

之后的测试可以用同局域网中的pc浏览器，输入板子的ip（192.168.X.X）

即可看到站点效果..
