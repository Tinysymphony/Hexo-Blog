Ubuntu Restoration

  • Backup
  • Ubuntu
  • linux
Ubuntu
近期不慎格式化全盘，大学以来所有的资料基本都丢失，配置完毕的Ubuntu也没有幸免。想了想还是把重新配置的过程再记一边，有备无患。

——————————————————————————————————————————

查看独显：

sudo su

cat /sys/kernel/debug/vgaswitcheroo/switch

(独显DIS DynOff则忽略，否则换驱动或者尝试命令切换显卡)

设置亮度，预设亮度，fn调节等

登陆wifi

ntpdate ntp.fudan.edu.cn

系统设置-语言-更新支持

添加语言：中日英

添加输入法：sunpinyin + anthy

允许第三方源

app store中获取：

GParted 分区工具

Chromium(红杏插件->登陆zjutiny@gmail)

Shutter 截图工具

Flash

FileZilla ftp传输工具

Dosbox

GIMP 图像处理软件

Amarok 音频播放器

Audacity 音频处理软件

下载软件：

wps (可选)

Jdk or Jre 1.8.0_25 (配置自行百度)

vmware workstation 11 (获取win和linux iso并安装)

idea 14 (先设置jdk)

eclipse java EE

android sdk 23.0.2

命令行获取：

git (设置完毕后，恢复project在~/workspace中)

vim (或者clone git 上已经配置完毕的版本)

build-essential

everpad

mysql-server

mysql-client

php5-mysql

php5

l2tp插件获取与设置

sudo apt-add-repository ppa:seriy-pr/network-manager-l2tp

sudo apt-get update

sudo apt-get install network-manager-l2tp-gnome

sudo service xl2tpd stop
sudo update-rc.d xl2tpd disable

设置默认有线的ip、dns等(确保ping通zju的10.5.1.5)

vpn-gate:10.5.1.5
添加vpn user-number+pwd (去掉PAP)
IPV4 自动获取地址(only)
添加域:10.5.1.5 / 10.5.1.9
添加routine

goagent
clone goagent
修改local文件夹中的appid
启动server文件夹upload.py输入id、gmail、pwd
成功后启动local文件夹中的goagent-gtk.py
chrome 可用SwitchySharp
命名为goagent
设置本地127.0.0.1 8087端口
勾选Use the same proxy server for all protocols
general选项中设置quick switch (添加direct connect和goagent)

GUI更改：

壁纸->makimakima~

unity-tweak-tool

docky

numix / numix icon
