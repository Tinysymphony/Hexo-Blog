Cross Compiling

配置树莓派交叉编译环境，可以使用git上的工具包：

git clone git://github.com/raspberrypi/tools.git

而后在~/.bashrc中加入路径：

PATH=$PATH:$HOME/Work/tools/arm-bcm2708/gcc-linaro-arm-linux-gnueabihf -raspbian-x64/bin

source .bashrc

这样就可以使用arm-linux-gnueabihf-gcc 进行编译。

——————————————————————————————————————————

原来是安装arm-linux-gnueabihf-gcc-4.8包的，利用-mcpu=xxx参数修改目标cpu从而交叉编译，能支持许多版本的cpu.

但一直没找到合适的，所以换了包。

ps: −mcpu=cortex−a7可适用于pcDuino
