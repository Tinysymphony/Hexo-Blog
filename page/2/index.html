Awk

看脚本的时候突然发现了一个貌似看到很多次但都没好好理解的指令awk

感觉用于文本分析还是比较实用的，做个笔记吧。

 

格式：awk ‘{pattern + action}’ {files}

pattern为正则，需要//

 

默认按行分割，而后以空格分割每行，或用制定符号分割

例如：

awk -F ‘/’ 表示以/分割

awk -F ‘:’ ‘{print $2,$3}’ 以:分割并输出每行的第二第三个参数，$0表示整行

awk ‘BEGIN {EXP1} {EXP2} END {EXP3}’ /filepath 对某文件分析前先执行EXP1，而后空格分割每行文本执行EXP2, 最后执行EXP3

awk ‘/reg1/{EXP1} /reg2/{EXP2} ’ /filepath 对某文件分割，若满足正则1则执行EXP1, 若满足正则2则执行EXP2

 

比较操作：

（正则比较使用~和！=）
awk ‘(compare1){EXP1} (compare2){EXP2}’ /filepath 满足比较1则执行EXP1,满足比较2则执行EXP2.

 

每一个项都会有判断和操作，若希望执行完当前忽略其他判断，可在{ }中加 ; next;

 

{ }中可以使用if(){ }else if(){ }else{ }、for(;;){}

print XX可以使用 printf “xx”, xx的形式替换

 

常用内置变量：
FILENAME

NR 处理过的文件行数，多文件叠加

FNR 当前文件处理过的行数

NF 每行分割后项数

FS 设置分隔符，即-F后的参数

Python Image | numpy Notes

说人话

今天在看数字水印的内容，还是数学苦手。不过尝试了下python的两个包，一个是图片处理的Image，还有方便数学运算的numpy（建议sudo apt-get install python-numpy获取）

Image库处理图片相当方便

import之后可以这样读写图片文件，同时可以根据后缀自动转换格式：

img=Image.open(‘xxx.png’)

img.save(‘newname.jpg’)

对于黑白图片是只有一个灰度的，而彩色RGB则是三个灰度，Image可以分别提取：

r,g,b=img.split()

获取大小：
lx, ly=img.size

提取的每一个原色灰度相当于一个黑白照片

Image可以使用匿名函数改变所有的像素点, 例如：

img.point( lambda p : p * 2 ) #增强每一个点的亮度

 

常用的变换稍微做个笔记

img.rotate(arc)

img.resize((x,y))

img.show()

img.crop((x1,y1,x2,y2))

img.paste(ImageObject, (x1,y1,x2,y2))

 

numpy配合Image能实现对图片数据的矩阵运算

（其实一维也能做二维的处理，因为想更直观偷懒所以试试看这个库）

最简单的获取方式：

picArray=numpy.array(img) #提取图片信息到二维矩阵

picArray.shape #查看其行列结构

picArray.size #项数量

picArray.ndim #维度

picArray.reshape(x,y) #重构矩阵，在列表转矩阵的时候很好用

picArray.tolist() #转列表

基本了解了下，高深的还不会Orz……

处理完后可以这样保存：
output=Image.fromarray(picArray)

output.save(‘xxxx.png’)

Write a bash shell to ping all the IPs that are available in the LAN.

The LAN IP in our room begins at 192.168.16.100, ends at 192.168.16.200.

The following program is to ping every possible ip and get its mac address (when the packet is send successfully), then match it with the given address of RPi.

 

Program ends when the RPi is found or every ip of the LAN is searched.

The ping result is redirected to ping.log .

 

 

 

Speaking of ntp server, modify the /etc/ntp.config

add server lists:

server ntp.fudan.edu.cn  prefer

server 0.asia.pool.ntp.org
 

The first server may be a better choice and I labeled it as “prefer”.

boardcast 192.168.16.255

NodeJs 3.22

Having a tutorial book about NodeJs, somewhat want to comprehand why it is so popluar in webservers.

Download the package from nodejs.org and make it, the make command may take 10mins to complete.

NodeJs is supported by NPM— Nodejs package manager, which is quite brilliant tool to manage projects, especially establishing dependencies.

Before building a project, we’d better create a package.json file in the root directory.

such as the following form:

{
“name”:”project name”,
“version”:”version code”,
“description”:”xxx”,
“dependencies”:{
“package A”:”A’s version”,
“package B”:”B’s version”,
… …
}
}

perform: npm install (under the directory where you put the pacage.json )
npm will get the packages needed from net and create a folder named node_modules to store them.
It’s easy to view all the packages downloaded by npm with command : npm ls
npm is also used to share projects.

Another thing I feel interesting is how a http service is started through node.
Create a js file using syntax of node.js, then start it in the terminal with command : node xxx.js (we can run it in background)
As the initial node.js code shown in the nodejs.org, a service is started at the specified port on the server.

您的板子——树莓派——已经上线..

8G SD 卡上已经装了系统，插入就好
通过串口连接PC..白绿黑三线分别对I4,I5,GND..推荐使用红线连接Power口供电，充电口配合三棒子的充电器经常接触不良。

因为PC是Ubuntu，所以安装了kermit用于接受串口输入
sudo apt-get install kermit

在不接电的情况下可以在/dev/目录下ls tty*以确定设备的名称，一般是ttyUSB0 或者 ttyUSB1
而后编辑主目录下的配置文件，修改要对接的设备名..
vim ~/.kermrc
这里只修改一行:set line /dev/ttyUSB0
之后在连上红线后，在pc terminal中输入：
sudo kermit （不造为什么一定要root权限求解释）
然后按下c即可接收来自板子的启动信息，强行刷屏

用hdmi连接寝室显示器也可以看到boot信息，默认不是启动图形界面，登陆后用命令：
sudo raspi-config
即可修改…（板子完全启动后，再连接显示器视频信号可能无法输出Orz）

推荐用PC调试板子，使用rasbian的图形界面需要很大耐心.
